Description détaillée

template<typename TPayoff, typename TAggregator>
class opt::Asian< TPayoff, TAggregator >

Option path-dépendante en Black–Scholes valorisée par Monte Carlo.

Paramètres du template
TPayoffType de payoff : double operator()(double agg, double ST) const
  • agg : valeur agrégée (moyenne, min, max, ...)
  • ST : valeur finale du sous-jacent à l'échéance
TAggregatorType d'agrégateur : double operator()(double agg, double price, double step) const

Variance reduction : variables antithétiques (optionnel). Grecques : bump-and-reprice (différences finies centrées).

◆ priceMC_BrownianBridge_Asymptotic()

template<typename TPayoff, typename TAggregator>
double opt::Asian< TPayoff, TAggregator >::priceMC_BrownianBridge_Asymptotic ( ) const

Prix asymptotique de référence avec correction Brownian Bridge (LOOKBACK ONLY).

Cette méthode calcule un prix numérique de référence en utilisant :

  • une correction Brownian Bridge pour l’estimation continue du minimum ou du maximum,
  • un nombre élevé de trajectoires Monte Carlo,
  • une discrétisation temporelle fine,
  • des variables antithétiques systématiquement activées.
Avertissement
Cette méthode est mathématiquement valable uniquement pour les options lookback, c’est-à-dire lorsque l’agrégateur représente un minimum ou un maximum. Elle n’est pas destinée à un usage opérationnel, mais à servir de valeur asymptotique de référence pour l’étude de convergence.
Renvoie
Prix Monte Carlo asymptotique (scalaire).

Références paths, opt::Option::R_, opt::Option::S0_, seed, opt::Option::sigma_, steps, opt::Option::T0_, et opt::Option::T_.
