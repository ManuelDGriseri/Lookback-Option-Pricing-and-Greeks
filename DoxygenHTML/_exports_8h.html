Description détaillée

Interface C (DLL) pour Excel/VBA – options lookback (floating strike).

Conventions :

  • Interface "extern C" pour compatibilité VBA.
  • Deux modes : (1) MC standard : suffixe _mc (2) MC avec réduction de variance (variables antithétiques) : suffixe _mc_vr
  • Pour chaque grandeur (prix, grecques) on expose : estimateur, erreur standard, borne basse IC, borne haute IC.
